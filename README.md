## Description

汤姆·里德尔的日记：一个交互式的哈利波特宇宙 AI 应用。

## Overview

**Riddle's Diary** 重现《哈利波特与密室》中的魂器日记本，与 16 岁的汤姆·里德尔对话，体验：

- 🎭 **双面人格**：温文尔雅的表象下隐藏着冷酷算计
- 🐍 **魔法世界沉浸**：霍格沃茨、斯莱特林、黑魔法元素
- 🧠 **心理操控**：通过引导性对话施加微妙影响
- 📚 **角色深度**：基于原著的性格特征、心理创伤和动机
